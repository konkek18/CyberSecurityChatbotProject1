Project Description:
This project is a Cybersecurity Awareness Chatbot developed using C#. The chatbot interacts with users and provides basic cybersecurity advice on topics such as passwords and phishing.

Features:
- User input (name and questions)
- Chatbot responses to cybersecurity topics
- Input validation (handles empty input)
- ASCII art user interface
- Audio greeting on startup
- Multiple classes (Program, Chatbot, User, AudioPlayer)

How It Works:
The program starts by playing an audio greeting and displaying ASCII art.  
The user enters their name and can then interact with the chatbot.  
The chatbot responds based on keywords such as "password" and "phishing".

How to Run:
1. Open the project in Visual Studio
2. Click Run
3. Enter your name
4. Ask questions like:
   - password
   - phishing
